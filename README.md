# Anthony Johnson · Portfolio Site

Personal portfolio for Anthony Johnson, Solutions Architect specializing in transit fare collection systems. Built as a single static HTML file, hosted on GitHub Pages.

## Structure

```
index.html                    The full portfolio (single page, no build step)
404.html                      Custom not-found page served by GitHub Pages
Johnson_Resume.pdf            Resume download
case-studies/_template.html   Template for future deep-dive case study pages
.nojekyll                     Tells GitHub Pages to serve files as-is
```
